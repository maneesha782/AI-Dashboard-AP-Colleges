# AI-Dashboard-AP-Colleges
AI Dashboard for Engineering Colleges in Andhra Pradesh Analysis

Overview
This project analyzes engineering colleges in Andhra Pradesh using Python and Google Colab.

Dataset Features:
College Name
District
Type
Branch
Fees Per Year
NAAC Grade
Placement Percentage

Tools Used:
Python
Pandas
Matplotlib
Google Colab

Project Features:
Data Cleaning
Exploratory Data Analysis
KPI Analysis
Data Visualizations
Insights and Conclusions
